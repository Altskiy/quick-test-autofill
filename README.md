# quick-test-autofill
Заполняет поля по атрибуту data-field, name или placeholder тестовыми данными - 
сгенерированными ИНН, КПП, ОГРН, ОГРНИП и т.д. и данными из словаря, такими как Имя, Фамилия, Отчество и т.д.
